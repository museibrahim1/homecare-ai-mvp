What protects the data along the way

The controls that sit inside every flow above.
LayerProtection
In transitHTTPS everywhere; WebSocket streams authenticated with the same JWT
At restPostgres with field-level Fernet encryption on sensitive fields (EIN, voiceprint); audio in access-controlled S3
IdentityJWT with short sessions, MFA support, bcrypt with password history, lockout after failed attempts, 15-minute web session timeout
AuthorizationRole checks server side; agency data isolation on every query; admin routes require admin role at the API, never the client
AbuseRate limits on auth and AI endpoints, plus per-tier AI limits (Starter 15, Growth 40, Enterprise 120 requests per minute)
PaymentsApple handles all card data; server-side JWS verification against Apple root certificates before any entitlement
ComplianceHIPAA audit logging middleware on every PHI route; BAAs and compliance docs in the repo; CI runs Trivy, Gitleaks, and CodeQL on every push
